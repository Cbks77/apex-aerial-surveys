## 🚀 Deployment

Hosted on **Hostinger** — connected to this GitHub repo for easy updates.

To update the site:
1. Edit `index.html`
2. `git add .`
3. `git commit -m "describe your change"`
4. `git push`
5. Hostinger auto-deploys from the main branch
